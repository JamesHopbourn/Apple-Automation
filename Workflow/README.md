# Worklfow
## [OmniFocus](https://workflow.is/workflows/20e2800391254d0c8e9e4672891d99a3)
用途：自动化添加来自剪切板链接到 OmniFocus，并在备注里添加附上链接。如果是微信链接会一同带上公众号名称与作者。

特点：
- 匹配并删除网页链接中多余网站的标题信息。
- 如果匹配到微信链接，添加完任务后会自动跳转回微信。
- 如果传入是文本的话，跳转到 Safari 并用谷歌进行搜索。
